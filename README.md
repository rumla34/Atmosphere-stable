# 大气层整合包系统稳定版，2023.12.10更新

2023.12.10更新StatusMonitor.ovl-1.0.2和DBI.nro--v652.如果你一直使用我发布的包，以后的更新中没有特别的提醒，都是直接覆盖就完成大气层文件的升级。

由鸭子风波导致DBI作者停更汉化版DBI.nro，此版本整合包的DBI.nro是647，其实不需要更新652，仅备份的英文版dbi.nro.en文件更新为DBI652，有需要的可以更名替换汉化版DBI647。

朗姆会新增极限超频插件覆盖包用于替换稳定包自带的是标准超频插件，非续航和Lite不建议使用极限超频插件。
